Randomised PCA

Randomized PCA is a variant of Principal Component Analysis (PCA) that uses randomization to improve computational efficiency. It is based on the idea of approximating the principal components of a matrix using random projections

Key Concepts

1. Random projection:A random matrix is used to project the data onto a lower-dimensional space.

2. Singular value Decomposition:SVD is computed on the projected data to extract the principal components.

3. Approximation:The resulting principal components are an approximation of the true principal components.

Theoretical Foundations

1. Johnson-Lindenstrauss Lemma:This lemma provides a theoretical foundation for random projections, showing that they can preserve the pairwise distances between points.

2. Random Matrix Theory:Random matrix theory provides a framework for understanding the properties of random matrices and their behavior in various applications.

Benefits

1. Computational Efficiency:Randomized PCA is faster than traditional PCA for large datasets

2. Scalability:Randomized PCA can handle large datasets that do not fit in memory

3. Flexibility:Randomized PCA can be used in a variety of applications, including real-time processing and distributed computing.

Limitations

1. Approximation Error:The approximation error of randomized PCA depends on the quality of the random projection and the number of components retained

2. Choice of Parameters:The choice of parameters, such as the number of components and the random matrix, can affect the performance of randomized PCA
